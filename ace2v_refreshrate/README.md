# ACE2V 刷新率覆写

通过覆写系统的刷新率文件达成调整应用、游戏刷新率的目标。

# 调整目标

| 应用名称 | 应用包名 | 应用Activity | 目标刷新率 | 备注 |
| --- | --- | --- | --- | --- |
| 初音未来：缤纷舞台 | com.hermes.mk | - | 120hz | 设置中开启，演出120帧，普通60帧 |
| Arcaea | moe.low.arc | - | 120hz | 原本就可以开120帧 |

## 使用须知

理论适配Magisk，测试适配Apatch。[获取文件地址](https://github.com/emptylight370/ace2v_refreshrate)
