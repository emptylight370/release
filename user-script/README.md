# Scripts

> [!NOTE]
> 以下脚本最初面向 Tampermonkey 开发，后续面向 ScriptCat 开发。各脚本最终支持 Tampermonkey 的版本见备注。在此版本后，可能使用到 ScriptCat 独占的功能，不保证兼容性。

## scnu sso

自动跳转到统一认证登录页面。  
目前支持：

- 砺儒云课堂
- 教务系统

> [!TIP]
> 最终支持 Tampermonkey 的版本为 1.1.0。

## scholat

目前支持学者网自动从主页进入登录页面

> [!TIP]
> 最终支持 Tampermonkey 的版本为 1.0.1。

## itzmx

支持自动填写itzmx论坛的签到框

> [!TIP]
> 最终支持 Tampermonkey 的版本为 1.0.0。

## Bangumi Enhanced

支持为Bangumi添加一些实用功能，比如复制标题按钮。

> [!TIP]
> 最终支持 Tampermonkey 的版本为 1.2.0。

## NexusMods

尝试去掉广告div。

> [!TIP]
> 最终支持 Tampermonkey 的版本为 1.3.0。

## gkd redirect

自动从官方的快照审查器跳转到第三方的快照审查器。默认从 `i.gkd.li` 跳转到 `li.chenge.eu.org`。  
如果选择跳转到自定义的第三方网站，则需要在脚本设置中添加作用域，以在目标站点显示脚本设置菜单。
